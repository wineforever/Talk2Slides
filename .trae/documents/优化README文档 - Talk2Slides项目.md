### 2. 新增"调试经验与常见问题"章节
基于实际调试经验，添加以下内容：

#### 2.1 视频生成成功但无输出文件问题
**现象**：系统提示视频生成成功，但在输出目录找不到文件
**原因**：
1. 时间轴数据无效（end <= start 或 duration <= 0）
2. FFmpeg处理失败但错误被吞掉
3. 输出目录权限问题
**解决方案**：
- 系统已添加严格的时间轴验证
- 增强的错误处理和阶段标记
- 详细的日志输出

#### 2.2 语义对齐失败问题
**现象**：DP算法失败，无法生成时间轴
**原因**：
1. PPT和字幕内容完全不相关
2. 相似度阈值设置过高（默认0.5）
3. 最小展示时长设置过长
**解决方案**：
- 系统已实现备用策略（均匀分布时间轴）
- 建议调整参数：相似度阈值→0.3，最小展示时长→1.0
- 增加调试日志和统计信息

#### 2.3 文件路径和权限问题
**现象**：文件处理失败，权限错误
**原因**：
1. 输出目录不可写
2. 临时目录空间不足
3. Windows路径编码问题
**解决方案**：
- 输出目录改为项目根目录下的`output/`
- 详细的路径验证和权限检查
- UTF-8路径编码处理

#### 2.4 依赖安装问题
**经验**：
- **FFmpeg安装**：Windows用户需要将ffmpeg添加到PATH环境变量
- **LibreOffice安装**：必须安装完整版，不仅仅是查看器
- **Python包冲突**：使用虚拟环境避免依赖冲突
